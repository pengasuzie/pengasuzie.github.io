with AI lead research.

A Claude Code skill that understands your product, researches target companies, and generates interactive HTML reports with fit scoring, digital maturity assessment, and personalized outreach strategies. One command, shareable output.

Get the Skill Setup Guide
How It Works
/lead-research → understand product → research companies → score & rank → interactive HTML report → push to CRM

Understands your business

Three modes for product context. Mode A: you describe your product and target market. Mode B: it reads your codebase to figure out what you're building. Mode C: both combined for the most accurate picture.

Smart inference picks the right mode automatically. If you're in a repo and provide a description, it suggests Mode C. It also checks for existing product context files before asking questions.

Scores and prioritizes

Each lead gets a fit score (1–10) based on ICP alignment, signals of need, budget indicators, competitive landscape, and timing. Scores map to tiers: HIGH (8–10), MEDIUM (5–7), LOW (1–4).

A separate digital maturity score (0–100%) estimates tech adoption from website quality, portals, TMS usage, API integrations, social presence, and online reputation.

Three Product Context Modes
Mode A

You describe it

Provide your value prop, target market, and differentiators directly in the prompt. Best when you have a clear pitch and want to control the framing.

Mode B

Codebase analysis

Reads your README, package files, CLAUDE.md, and marketing pages to figure out what you're building. Presents its assessment for confirmation before proceeding.

Mode C — Recommended

Both merged

Combines your description with codebase analysis. Flags any differences between what you say and what the code shows. The most accurate targeting.

Interactive HTML Reports

Shareable dashboards

Output is a self-contained HTML file with embedded CSS and vanilla JS. No dependencies, no build step. Open it in a browser, share it with your team, or email it to a co-founder.

  • Stats bar with totals and tier counts
  • Full-text search across all cards
  • Tag-based filter buttons (score, industry, certs, size)
  • Company cards with scoring, contacts, and outreach angles
  • Summary tables: top prospects ranked, tech landscape
  • Inactive/merged companies shown with context

Data Per Lead

Company intelligence

  • Company name, website, phone, location
  • Founded year and company size
  • Industry, sector, and services offered
  • Certifications and accreditations
  • Current software and TMS platforms
  • ABN or company registration number

Contact strategies

  • Multiple key people with titles
  • LinkedIn URLs for decision-makers
  • Personalized outreach strategy per lead
  • Conversation starters and value propositions
  • Digital maturity assessment (0–100%)
  • HubSpot CRM integration (push leads directly)
Setup Guide
01

Get the skill file

Clone the skills repo (or just download the single file) and copy it into your Claude Code skills directory.

git clone https://github.com/pengasuzie/claude-skills.git mkdir -p ~/.claude/skills/lead-research-assistant cp claude-skills/lead-research-assistant/SKILL.md \ ~/.claude/skills/lead-research-assistant/
02

Use it

Open Claude Code and run the skill. Describe your product and target market, or let it read your codebase. It handles the rest.

claude # Mode A — you describe it > /lead-research-assistant I'm building a compliance tool for freight forwarders. Find 15 prospects in Brisbane. # Mode B — codebase analysis > /lead-research-assistant Look at this repo and find 15 companies in Melbourne that would be customers. # Mode C — both (recommended) > /lead-research-assistant We're a trade compliance platform — check the codebase too. Find 20 prospects in Sydney.
03

Get your report

The skill generates a self-contained HTML file. Open it in your browser to search, filter, and review leads. Share it with anyone — no dependencies needed.

Researching target companies... Found 18 matching companies. Scoring and enriching leads... Report saved: lead-research-brisbane-freight-2026-02-27.html Summary: 6 HIGH (8-10) — ideal prospects 7 MEDIUM (5-7) — possible fit 2 LOW (1-4) — already using competitor Want me to add these to HubSpot CRM?
After the Research

CRM integration

With the HubSpot MCP server connected, the skill can push leads directly into your CRM. It creates Company records, Contact records for decision-makers, and optionally creates Deals in your pipeline. Always confirms before creating anything.

Without HubSpot, it falls back to CSV export.

Cold outreach

Pairs with the cold-email skill for follow-up. The lead research provides the targeting and personalization data, then cold-email generates outreach sequences tailored to each prospect's situation.

The outreach strategy section in each lead card gives you conversation starters and value propositions ready to use.
